# GUTBRAINS

Information Extraction for the GutBrainIE task at CLEF 2026. The repo contains baseline implementations, LLM-based pipelines, and utilities to preprocess data, run experiments, and evaluate predictions for entity and relation extraction in the gut-brain domain.

## Authors

- Bita Khashechian
- Danil Smirnov

## Repository layout

- `baselines/` — baseline models and reference implementations
- `runs/` — experimental runs and scripts
- `src/` — core pipeline code and utilities
- `data/` — dataset placeholders and processing scripts
- `results/` — predictions, logs, and evaluation outputs

## Task

The project targets the GutBrainIE challenge, which focuses on extracting entities and relations from PubMed abstracts about the gut-brain interplay.

## Notes

Training data are provided by the shared task organizers. This repository is intended for research and experimental purposes.
